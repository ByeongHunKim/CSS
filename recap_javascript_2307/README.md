## for loop

| 종류                                   | 동작구조                                           |
| -------------------------------------- | -------------------------------------------------- |
| `for .. in`                            | `key 값만 순회한다`                                |
| `for .. of`                            | `value 값만 순회한다`                              |
| `while문`                              | `while문 안에 있는 조건이 참일 때까지만 순회한다`  |
| `break문이 for loop 안에 있는 경우`    | `순회가 종료된다`                                  |
| `continue문이 for loop 안에 있는 경우` | `현재 루프만 종료 후 다음 루프도 이동(종료x스킵o)` |
